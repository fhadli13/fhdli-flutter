```

Framework • revision 5dcb86f68f (7 days ago) • 2024-05-09 07:39:20 -0500
```

#### Build Application

```
·]>>  git clone https://github.com/toly1994328/FlutterUnit.git
·]>>  cd FlutterUnit

Build Android:
·]>>  flutter build apk --target-platform --split-per-abi
Build iOS:
·]>>  flutter build ios
Build Windows:
·]>>  flutter build windows  
Build Linux:
·]>>  flutter build linux
Build web:
·]>>  flutter build web
```
